ComfyUI配置z-image-turbo工作流生成图片

2026年03月05日 · 1 分钟 · 159 字 · 八戒

书接上文，操作系统是debian 12，搭配AMD 6700 xt 12G的显卡

驱动安装看这篇：

然后我们来安装

# 翻墙才可以clone项目
export https_proxy=http://192.168.1.1:1080
git clone https://github.com/Comfy-Org/ComfyUI

# 安装uv
curl -LsSf https://astral.sh/uv/install.sh | sh
export PATH="$HOME/.local/bin:$PATH"

# 安装python 3.12，比较保险
uv python list
uv python install 3.12


cd ComfyUI/

# 初始化
uv init
uv venv --python 3.12
. .venv/bin/activate

# 安装 torch ，要选择合适的torch和rocm 6.0
uv pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/rocm6.0
uv pip list

# 安装附加
uv pip install -r requirements.txt

ComfyUI就装好了

然后利用魔搭下载大模型，魔搭国内，我们就不用跑到hugingface去下了

# 关掉proxy
unset https_proxy

# 安装魔搭
uv pip install modelscope

# 新建下载临时目录
mkdir tmp
cd tmp

# 下载，local_dir都是 ./

# 7.49G
modelscope download --model Comfy-Org/z_image_turbo split_files/text_encoders/qwen_3_4b.safetensors --local_dir ./

# 320M
modelscope download --model Comfy-Org/z_image_turbo split_files/vae/ae.safetensors --local_dir ./

# 11.5G
modelscope download --model Comfy-Org/z_image_turbo split_files/diffusion_models/z_image_turbo_bf16.safetensors --local_dir ./

# 4.20G
modelscope download --model Comfy-Org/z_image_turbo split_files/diffusion_models/z_image_turbo_nvfp4.safetensors --local_dir ./

上面我们下了z_image_turbo的bf16和nvfp4两个模型，看自己的显卡哪个能用就用哪个了。

下载完成后，我们会得到4个文件

然后我们要把文件放到ComFyUI/models目录下，模型的文件都统一放在在这里面

mv split_files/diffusion_modes/* ComfyUI/models/diffusion_models

mv split_files/vae/* ComfyUI/models/vae

mv text_encoders/split_files/text_encoders/qwen_3_4b.safetensors ComfyUI/models/text_encoders

然后我们在ComFyUI目录下执行主程序

cd ComfyUI

export HSA_OVERRIDE_GFX_VERSION=10.3.0
uv run main.py --listen 0.0.0.0

然后我们打开网页 http://ip:8188/

下载、打开加载我们的工作流文件：
